Задания:

1. Написать программу, которая выводит на экран шахматную доску с заданным размером клетки.

2. Пользователь вводит целое число. Необходимо вывести все целые числа, на которое заданное число делится без остатка.

3. Пользователь вводит два целых числа. Необходимо вывести все целые числа, на которые оба введенных числа делятся без остатка.

4. Пользователь вводит целое число А. Необходимо вывести все целые числа В, где А делится без остатка на В*В и не делится без остатка на В*В*В 

5. Пользователь вводит число. Определить количество цифр в этом числе, посчитать их сумму и среднее арифметическое. Определить количество нулей в этом числе. Общение с пользователем организовать через меню. (ПОВТОР)
