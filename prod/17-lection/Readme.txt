Задания:

1. Написать программу, которая подсчитывает колличество »Счастливых чисел» в массиве. Счастливым считается число в котором сумма цифр делится на 3

2. Написать программу, копирующую элементы 2-х массивов размером 5 элементов каждый в один массив размером 10 элементов следующим образом: сначала копируются последовательно все элементы, большие 0, затем последовательно все элементы, равные 0, а затем последовательно все элементы, меньшие 0

3. Написать функцию которая принимает входной массив, число M и выходной массив, в который заполняется только элементами первого массива которые делятся на М

4. Написать функцию, которая принимает массив. За один обход массива поменять местами соседние элементы, в которых элемент слева больше правого

5. Модифицировать последнюю функцию таким образом, чтобы она делала N-1 проходов, где N – колличество элементов в массиве. Реализовать выход из функции в момент, когда за проход не было сделана ни одного перемещения. (Совмещено с четвёртым заданием)
