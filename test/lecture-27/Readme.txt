Задания:

1. Реализовать структуру «Стиральная машинка» (фирма, цвет, ширина, длина, высота, мощность, 
скорость отжима, температура нагрева). Создайте экземпляр структуры и проиллюстрируйте 
работу с ним. Используйте перечисления.

2. Реализовать структуру «Утюг» (фирма, модель, цвет, минимальная температура, максимальная 
температура, подача пара да/нет, мощность). Создайте экземпляр структуры и проиллюстрируйте 
работу с ним. Используйте перечисления.

3. Реализовать структуру «телефон» (фирма, цвет, экран, объем памяти, частота процессора). 
Создайте экземпляр структуры и проиллюстрируйте работу с ним. Используйте перечисления.

4. Разработать структуру, которая описывает животное (название, класс, кличка). Создать 
функции для работы с этой структурой:
  
  - Заполнение объекта, например несколькими функциями «создать кота», «создать собаку»
  - Вывод на экран данных об объекте
  - Функция «Подать голос»
