Задания:

1. Дан текстовый файл. Подсчитать количество слов, начинающихся с символа, который 
задаёт пользователь.

2. Дан текстовый файл. Переписать в другой файл все его строки с заменой в них 
символа 0 на символ 1 и наоборот.

3. Дан массив строк. Записать их в файл, расположив каждый элемент массива на 
отдельной строке с сохранением порядка.

4. Дан текстовый файл. Подсчитать количество символов в нём.

5. Дан текстовый файл. Подсчитать количество строк в нём.

6. Дан текстовый файл. Вывести самую длинную строку.

7. Дан текстовый файл. Найти и заменить в нём заданное пользователем слово.

8. Дан текстовый файл. Подсчитать сколько раз в нём встречается заданное 
пользователем слово (слово «из» в файле с содержимым «из изгороди» встречается 
один раз).

9. Дано два текстовых файла. Выяснить совпадают ли в них строки, вывести строки 
которые не совпадают на экран.
