Задания:

1. реализовать все алгоритмы (вставкой, обменом, пузырьком, быстрая сортировка)

- Вставкой: первый элемент неотсортированной части смещается на своё место в отсортированной части путём сравнения с стоящим до него.

- Обменом: в неотсортированной части находится индекс минимального числа, затем это число меняется местами с первым числом в неотсортированной части, становясь максимальным в отсортированной части.

- Пузырьком: по массиву проводится N-1 пробегов, где N - это длинна массива, при каждом пробеге сравниваются соседние пары чисел и меняются местами, если они не соответствуют заданным параметрам (больше, меньше)

- Быстрая сортировка: Находим «опорный» элемент, слева от него располагаем элементы меньше, справа элементы больше, повторяем рекурсивно для правой и левой части

2. модифицировать любой алгоритм сортировки так, чтобы он работал с массивом в определённом формате: в массиве хранятся пары чисел, левое является ключом, правое – соответствующим ему значением. Сортировку осуществить по ключу. Пары нельзя разрывать при сортировке. Таким образом ключ n-го элемента находится по индексу N*2, а значение по индексу N*2+1
